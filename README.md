# AI-MADNESS-
Source Code Of AI Madness

# How to Use This Code

Clone this Repo by git
```
git clone https://github.com/Jackpk52/AI-MADNESS-
```

After Cloning Go to the directory

Open .env and fill in all your details
then in the same directory do this in Command Prompt
```
npm install
```

When your ready to start the bot just do this in command prompt (again)

```
node run.js
```
